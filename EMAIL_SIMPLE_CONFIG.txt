=== EMAIL SETUP SEDERHANA ===

PILIH SALAH SATU KONFIGURASI:

1. GMAIL (RECOMMENDED)
========================
Edit file .env, ubah bagian email menjadi:

MAIL_MAILER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=pusdatinppkp@gmail.com
MAIL_PASSWORD=zrzgmnmjnjjfppcl
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS="pusdatinppkp@gmail.com"
MAIL_FROM_NAME="Sistem MCU"

Setup Gmail App Password:
1. Buka Google Account Settings
2. Security → 2-Step Verification (aktifkan)
3. App passwords → Generate password untuk "Mail"
4. Gunakan App Password sebagai MAIL_PASSWORD

2. OUTLOOK/HOTMAIL
==================
MAIL_MAILER=smtp
MAIL_HOST=smtp-mail.outlook.com
MAIL_PORT=587
MAIL_USERNAME=your-email@outlook.com
MAIL_PASSWORD=your-password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS="your-email@outlook.com"
MAIL_FROM_NAME="Sistem MCU"

LANGKAH SETUP:
==============
1. Edit file .env dengan salah satu konfigurasi di atas
2. Ganti email dan password sesuai kebutuhan
3. Save file .env
4. Run: php artisan config:clear
5. Test: php artisan email:test your-email@example.com

TEST EMAIL:
===========
- Test basic: php artisan email:test your-email@example.com
- Test MCU invitation: php artisan mcu:send-invitations --type=email
- Via admin panel: /admin/schedules → Send Invitation

TROUBLESHOOTING:
================
- Error "Authentication failed" → Gunakan App Password untuk Gmail
- Error "Connection failed" → Cek host dan port
- Email masuk spam → Setup SPF/DKIM records
